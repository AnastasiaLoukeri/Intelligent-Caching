This repository provides a Reinforcement Learning (RL) framework designed to optimize cache replacement policies. By treating cache eviction as a sequential decision-making problem, the project trains intelligent agents using Proximal Policy Optimization (PPO) to dynamically outperform traditional heuristics like LRU and LFU. The framework includes a custom cache simulator with a tailored reward function, synthetic Zipf-distributed workload generation, and several PyTorch-based neural network architectures—ranging from basic feedforward networks to Convolutional Neural Networks (CNNs) and Transformers. Ultimately, the system learns workload-specific data eviction strategies to minimize cache miss rates and improve overall memory access efficiency.
